# Reel Forge - HTML/CSS/JS Version

This is the plain HTML, CSS, and JavaScript version of the Reel Forge portfolio, optimized for quick deployment on Vercel.

## Features
- **Premium Design**: Modern aesthetics with glassmorphism and gold accents.
- **Dynamic Portfolio**: Filterable video portfolio with custom modal player.
- **Fast Performance**: No frameworks, just vanilla code for lightning-fast loads.
- **Vercel Ready**: Includes `vercel.json` for easy deployment.
- **Backend Connected**: Ready to connect to your Express/Node.js backend.

## Deployment
1. Initialize a git repository: `git init`
2. Add all files: `git add .`
3. Commit: `git commit -m "Initial commit"`
4. Push to GitHub/GitLab/Bitbucket.
5. Connect your repository to Vercel.

## Development
To run locally, just open `index.html` in your browser or use a live server extension.

## Backend
The backend is located in the `/backend` folder. You can deploy it separately or as a monorepo. Ensure the `apiUrl` in `script.js` points to your deployed backend URL.
